బీఆర్ఎస్ అధినేత కేసీఆర్ (KCR) ప్రజల్లోకి వెళ్తే మంత్రులకు నిద్ర పట్టడం లేదని మాజీమంత్రి, సిద్దిపేట ఎమ్మెల్యే హరీష్‌రావు (Harish Rao) అన్నారు. శనివారం నాడు జహీరాబాద్ లో బీఆర్ఎస్ కార్యకర్తల సమావేశం నిర్వహించారు. ఈ సమావేశంలో హరీష్‌రావు పాల్గొని కాంగ్రెస్ ప్రభుత్వంపై తీవ్ర విమర్శలు గుప్పించారు

హైదరాబాద్: బీజేపీ తెలంగాణ రాష్ట్ర అధ్యక్షుడు బండి సంజయ్ చేపట్టిన ప్రజా సంగ్రామ యాత్రపై టీఆర్ఎస్ వర్కింగ్ ప్రెసిడెంట్, మంత్రి కేటీఆర్ విమర్శలు గుప్పించారు. బండి సంజయ్‌కు పాలమూరులో అడుగుపెట్టే అర్హత లేదని, ఆయన చేపట్టినది ముమ్మాటికీ ప్రజా వంచన యాత్ర అని విమర్శించారు. పాలమూరు రైతులకు ద్రోహం చేస్తూ యాత్రలు చేస్తారా? అని కేటీఆర్ నిలదీశారు
